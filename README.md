# Organizador Financeiro com IA

Este projeto tem como objetivo ajudar pessoas a organizarem seus gastos mensais
utilizando automação simples em Python e apoio de IA.

## O que este projeto faz
- Organiza despesas mensais
- Gera resumo de gastos
- Ajuda na tomada de decisão financeira

## Tecnologias utilizadas
- Python
- IA (prompts)
- Planilhas
